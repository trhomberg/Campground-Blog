# Campground-Blog

A simple blog app created with NodeJS, MongoDB, and Javascript.

### Technologies Used

* HTML/CSS
* Javascript
* NodeJS
* MongoDB

#### Dependencies

* NodeJS Version: 14.15.4
* ExpressJS Version: 4.17.1
* PassportJS Version: 0.4.1
* Ejs Version: 3.1.5
* Mongoose Version: 5.10.5
